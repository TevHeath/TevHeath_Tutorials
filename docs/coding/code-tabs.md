## Content Tabs

This is some examples of content tabs.

### Generic Content

=== "Plain text"

    This is some plain text

=== "Unordered list"

    * First item
    * Second item
    * Third item

=== "Ordered list"

    1. First item
    2. Second item
    3. Third item

### Code Blocks in Content Tabs

=== "Python"

    ```py
    def main():
        print("Hello world!")

    if __name__ == "__main__":
        main()
    ```

=== "MATLAB"

    ```js
    function main() {
        console.log("Hello world!");
    }

    main();
    ```

=== "C++"

    ```cpp
    // C++ code example
    #include <iostream>
    int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
    }
    ```
